Aquí puedes guardar tus propios iconos e imágenes personalizadas (armas, cascos, etc.). 
Como estamos usando Next.js, cualquier archivo que pongas en esta carpeta estará disponible para la aplicación.

Por ejemplo, si guardas un archivo llamado "cinetica.png" aquí,
lo podremos usar en el código escribiendo la ruta: "/custom-icons/cinetica.png"
